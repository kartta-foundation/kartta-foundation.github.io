Our Services

Kartta Foundation develops advanced machine learning tools for historical map processing, geospatial AI, and semantic extraction.

  github.com/kartta-foundation

mapKurator

Automated pipeline for detecting, extracting, and geocoding text from scanned historical maps.

  • Paper: The mapKurator System: A Complete Pipeline for Extracting and Linking Text from Historical Maps

PALETTE

End-to-end text spotting model for historical maps, supporting Latin (English), Tranditional Chinese, and Japanese.

  • Paper: Hyper-local deformable transformers for text spotting on historical maps

LIGHT

Multi-modal text linking model that fuse text's spatial structure, visual cues, and linguistic context to group map text into meaningful phrases.

  • Paper: LIGHT: Multi-modal Text Linking on Historical Maps
